# Text Embeddings

This folder contains scripts for generating biomedical text embeddings for the
evidence sentences.

## Contents

- `generate_embeddings_biobert.py`: generate BioBERT/MedCPT-style embeddings.
- `generate_embeddings_pubmedbert.py`: generate PubMedBERT/MedCPT-style
  embeddings.

## Expected Outputs

```text
text_embeddings/
├── embeddings_biobert_mcpt.npy
├── embeddings_pubmedbert_mcpt.npy
└── index.tsv
```
